# Architects Office SP - Site Institucional

Este é um site institucional "white-hat" para a empresa ARCHITECTS OFFICE SP ARQUITETURA LTDA, CNPJ 32.224.852/0001-59, localizada em São Paulo, SP, com atividade principal de Arquitetura e Urbanismo.

## Estrutura do Projeto

- `index.html`: Página principal com seções de herói, sobre, projetos e contato.
- `css/style.css`: Folha de estilos unificada com design responsivo, paleta de cores e tipografia moderna.
- `js/script.js`: Arquivo JavaScript para rolagem suave, menu mobile e interações de formulário.
- `images/`: Pasta vazia, pois todas as imagens são carregadas via URLs externas.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript puro

## Design e Funcionalidades

- **Design Responsivo:** O site é totalmente responsivo e se adapta a diferentes tamanhos de tela (desktop, tablet, mobile).
- **Tipografia Premium:** Utiliza fontes como Montserrat e Raleway para uma estética profissional e temática de arquitetura.
- **Redirecionamento de Links:** Todos os botões e imagens são configurados para redirecionar para `https://google.com` para fins de aquecimento de domínio.
- **Formulário de Contato:** Um formulário de contato simples com simulação de envio (sem backend).

## Como Editar e Implantar

1.  **Conteúdo:** Edite o `index.html` para alterar textos, títulos e links.
2.  **Estilos:** Modifique `css/style.css` para ajustar cores, fontes e layout.
3.  **Scripts:** O `js/script.js` contém funcionalidades básicas. Evite modificações complexas se não tiver familiaridade com JavaScript.
4.  **Imagens:** As imagens são carregadas via URLs externas. Para alterar, substitua as URLs nos atributos `src` das tags `<img>` no `index.html`.
5.  **Implantação:** Faça o upload de todos os arquivos e pastas para um servidor web. O site está pronto para ser hospedado em qualquer serviço de hospedagem estática.
